
# Python Web Scraper

A simple web scraper built in Python that extracts book data from an ecommerce store.

## Features

- Scrapes book listings from ecommerce platforms
- Extracts book titles, prices, and ratings
- Saves data to CSV or JSON format
- Handles basic error handling and retries

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python scraper.py
```

## Requirements

- Python 3.7+
- requests
- beautifulsoup4

## Output

Scraped data is saved to `books.csv` or `books.json` in the project directory.

## Legal Notice

Ensure you have permission to scrape the target website and comply with their `robots.txt` and terms of service.
